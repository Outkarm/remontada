## 📖 About the Project <a name="about-the-project"></a>

### 🎯 Purpose <a name="purpose"></a>

**KoyOutkarm Portfolio** is a full-stack web application designed to showcase the developer's personal projects, skills, and AI-enhanced workflows. It serves as a dynamic portfolio for potential employers, collaborators, or clients to interact with and understand the developer's capabilities.

## 💻 Getting Started <a name="getting-started"></a>

Follow the instructions below to set up the project locally.

---

### ⚙️ Backend Setup <a name="backend-setup"></a>

#### Prerequisites

Make sure you have the following installed:

- Ruby
- Ruby on Rails
- PostgreSQL

#### Clone Backend Repo:

```bash
git clone https://github.com/Outkarm/remontada-api.git
cd remontada-api
```

#### Install Dependencies:

```bash
bundle install
```

#### Setup Database:

```bash
rails db:create
rails db:migrate
```

#### Start Backend Server:

```bash
rails server
```

---

### 🎨 Frontend Setup <a name="frontend-setup"></a>

#### Prerequisites

- Node.js
- npm

#### Clone Frontend Repo:

```bash
git clone https://github.com/Outkarm/remontada.git
cd remontada
```

#### Install Dependencies:

```bash
npm install
```

#### Set up environment variable:

```bash
VITE_API_URL=http://your-backend-url.com
VITE_PASSWORD=your-password-hash
```

#### Start Development Server:

```bash
npm run dev
```

> 💡 You may need to update CORS settings on the backend if accessing from different ports during development.

---

### 🚢 Deployment <a name="deployment"></a>

You can deploy this project using platforms like **Render**, **Vercel**, or **GitHub Pages** (frontend only).

- **Render (Full-stack):** Configure both frontend and backend services.
- **Vercel (Frontend only):** Connect GitHub and set build command to `npm run build`.
